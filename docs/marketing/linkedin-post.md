# 링크드인 포스팅 초안

## 포맷: 스토리텔링형 (직장인 공감)

---

아침 6시 반. 카페에서 하루를 시작하고 싶었습니다.

"강남 새벽 카페" 검색.
카카오맵에 뜨는 카페를 하나씩 눌러봅니다.
영업시간 확인 → 9시 오픈. 다음.
영업시간 확인 → 정보 없음. 다음.
영업시간 확인 → 10시. 다음.

매번 이랬습니다.

그래서 주말에 직접 만들었습니다.

"모닝커피" — 서울에서 아침 일찍 여는 카페만 모아놓은 지도.

- 3,000개 이상의 얼리버드 카페 데이터
- 6시/7시/8시 시간대별 필터
- 요일별 영업시간 (주말 다른 카페도 정확히)
- 25개 구별 필터
- 카카오톡 공유로 동료에게 바로 추천

기술적으로는 Next.js + Supabase + 카카오맵 SDK로,
뷰포트 기반 마커 렌더링으로 3000개 카페도 버벅임 없이 표시합니다.

"아침 카페를 찾아 출발하기까지 걸리는 시간"을 줄이는 것.
이게 이 서비스의 유일한 목표입니다.

써보시고 피드백 주시면 감사하겠습니다.

## 포맷 가이드

### 구조
1. **훅** (첫 2줄): 공감할 수 있는 상황 묘사
2. **갈등**: "매번 이랬습니다" — 불편함 강조
3. **해결**: "직접 만들었습니다" — 전환점
4. **기능 나열**: 불릿 포인트 5~6개 (스캔 가능하게)
5. **기술 한 줄**: 개발자 네트워크를 위한 크레딧
6. **한 줄 미션**: 감성적 마무리
7. **CTA**: 링크 + 피드백 요청
8. **해시태그**: 5~7개

### 이미지
- 슬라이드 1: 지도 전체 뷰 (마커 색상 눈에 띄게)
- 슬라이드 2: 필터 칩 + 바텀시트 상세
- 슬라이드 3: 카카오톡 공유 카드 미리보기

### 게시 팁
- 화~목 오전 8~9시 (출근길 스크롤 시간)
- 첫 1시간 댓글/반응에 빠르게 응답 (알고리즘 부스트)
- 팀원/동료 5~10명에게 리포스트 부탁 (초기 도달 확보)
- "출근 전 카페 가시는 분?" 질문형 댓글 유도

### 팔로업 포스트 (1~2주 후)
- "모닝커피 만든 지 2주, 데이터로 본 서울 아침 카페"
- GA4 데이터 기반: 가장 많이 검색된 구, 인기 시간대, 공유 수
- 숫자 중심 → 링크드인에서 데이터 포스트 반응 좋음

### 주의
- 자랑 톤 금지. "만들었습니다" > "대박 나고 있습니다"
- 기술 과시 금지. 기술은 1줄만. 사용자 가치 중심
- 겸손한 피드백 요청이 공감 + 댓글 유도
