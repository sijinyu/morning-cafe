# 인스타그램 카드뉴스 기획안

## 콘텐츠 시리즈

### 시리즈 1: "구별 아침 카페 TOP 5"
DB 데이터 기반으로 각 구의 가장 이른 오픈 카페를 추출.

- **1화**: 강남구 6시 오픈 카페 TOP 5
- **2화**: 마포구 아침 카페 추천 TOP 5
- **3화**: 성동구(성수동) 새벽 카페 TOP 5
- **4화**: 서초구 아침 카페 TOP 5
- **5화**: 영등포구(여의도) 아침 카페 TOP 5
- **확장**: 송파구, 종로구, 용산구 등

**카드 구성** (5장):
1. 커버: "{구}에서 아침 가장 빨리 여는 카페 🌅" + 모닝커피 로고
2~4. 각 카페 (이름, 오픈시간, 주소, 한줄 코멘트)
5. CTA: "더 많은 아침 카페는 morningcoffee.kr에서"

### 시리즈 2: "아침형 인간 카페 루틴"
스토리텔링형 콘텐츠.

- "5시 반 기상 → 6시 카페 도착 → 7시 출근" 타임라인
- 실제 6시 오픈 카페 예시와 함께
- "아침 카페에서 할 수 있는 것" (독서, 공부, 작업, 조식)

### 시리즈 3: "이번 주말 아침 카페"
주말에 여는 카페 중심 (평일과 다른 오픈시간).

- "토요일 8시 이전 오픈하는 카페"
- 브런치 가능한 곳 위주
- 주차 가능 카페 표시

### 시리즈 4: "24시간 카페 지도"
830개 24시간 카페 데이터 활용.

- "새벽 작업하기 좋은 24시간 카페"
- 구별 24시간 카페 분포

## 데이터 쿼리 (카드뉴스 소재)

구별 TOP 5 추출 예시:
```sql
SELECT name, address, opening_time, phone
FROM cafes_with_coords
WHERE is_earlybird = true
  AND address ILIKE '%강남구%'
  AND opening_time IS NOT NULL
ORDER BY opening_time ASC
LIMIT 5;
```

이 쿼리를 `scripts/` 폴더에 스크립트화해서 구별로 자동 추출 가능.
