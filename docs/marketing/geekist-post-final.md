# 지피터스 최종 포스팅

## 제목
주말에 만든 "서울 얼리버드 카페 지도" — 9,400개 카페 데이터로 아침을 바꿨습니다

## 본문

### 만들게 된 계기

아침 일찍 카페에서 하루를 시작하고 싶은데, 매번 "강남 새벽 카페", "성수동 아침 카페"를 검색하는 게 너무 비효율적이었습니다.

카카오맵으로 검색해도 **영업시간 필터가 없어서**, 지도에 뜨는 카페를 하나하나 열어보며 "이 카페 7시에 여나?" 확인해야 했습니다.

"아침 일찍 여는 카페만 모아서 지도로 보여주면 어떨까?"

주말에 만들기 시작했고, 지금 바로 쓸 수 있습니다.

### 핵심 기능

**1. 시간대 필터**
~6시, 6~7시, 7~8시 — 원하는 오픈 시간대만 한 탭에 필터링

[스크린샷 2: 필터 칩 바 — 시간/요일/구 필터]

**2. 요일별 영업시간**
"이 카페 토요일에도 7시에 여나?" — 주말과 평일 오픈 시간이 다른 카페도 정확히 표시합니다.

**3. 구별 필터**
강남구, 마포구, 성동구 등 25개 구별로 빠르게 좁히기

**4. 체인/개인 필터**
"스타벅스 말고 개인 카페만 볼래" — 체인 숨기기 한 탭

**5. 상세 정보**
사진, 메뉴, 별점, 주차, 편의시설을 바텀시트에서 한눈에

[스크린샷 3: 바텀시트 상세 — 사진 캐러셀 + 정보]

**6. 카카오톡 공유**
발견한 카페를 친구/동료에게 카카오톡으로 바로 공유. OG 이미지까지 예쁘게 나옵니다.

[스크린샷 4: 카카오톡 공유 카드]

**7. PWA**
홈 화면에 추가하면 네이티브 앱처럼 사용 가능

### 만들면서 배운 것

**1. 3,000개 마커 렌더링**
전체 마커를 한번에 그리면 메모리 폭발. 뷰포트 기반 필터링으로 화면에 보이는 50~200개만 렌더링하니까 부드럽게 돌아갑니다.

**2. 요일별 영업시간의 함정**
`hours_by_day`에서 해당 요일 키가 없으면 "정보 없음"으로 처리해야 하는데, 기본 `opening_time`으로 fallback하는 버그가 있었습니다. 토요일 9시 오픈 카페가 평일 7시 필터에 잡히던 문제를 한참 헤매다 발견했습니다.

**3. 카카오 CDN 이미지 프록시**
카카오맵 사진 중 네이버 호스팅 이미지가 Referer 제한으로 깨지는 문제 → kakaocdn cthumb 프록시로 우회. 이미지 최적화 + 캐싱까지 한번에 해결.
