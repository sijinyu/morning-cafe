# 링크드인 최종 포스팅

## 본문 (복사해서 바로 게시)

---

아침 러닝 끝나고 분위기 있는 카페 가고 싶을 때 있잖아요.
네이버 지도에서 시간 필터 걸면 — 스타벅스, 메가커피, 이디야만 나옵니다.

"프랜차이즈 빼고 아침에 여는 개인 카페만 보고 싶은데?"

그래서 주말에 직접 만들었습니다.
"모닝커피" — 서울 아침 카페 지도.

• 3,000개 얼리버드 카페
• 6시 / 7시 / 8시 시간대별 필터
• 프랜차이즈 on/off (이게 핵심)
• 25개 구별 필터
• 요일별 영업시간 (주말 다른 카페도 정확히)

Next.js + Supabase + 카카오맵 SDK.
뷰포트 기반 마커 렌더링으로 3,000개도 버벅임 없습니다.

써보시고 피드백 주시면 감사하겠습니다.

👉 https://morning-cafe-phi.vercel.app

## 이미지 (캐러셀 형태로 첨부)

1. 지도 전체 뷰 — 프랜차이즈 off 상태에서 개인 카페만 보이는 화면
2. 필터 칩 + 바텀시트 상세
3. 카카오톡 공유 카드

## 게시 가이드

- 타이밍: 화~목 오전 8~9시 (출근길 피드 노출)
- 첫 1시간: 댓글/반응에 빠르게 응답
- 리포스트 부탁: 동료 5~10명에게 미리 연락
- 첫 댓글: "출근 전 카페 가시는 분 계신가요?" 달기

## 팔로업 포스트 (2주 후)

"모닝커피 만든 지 2주, 데이터로 본 서울 아침 카페"
