# 채널별 마케팅 체크리스트

## 우선순위 1: 즉시 실행 (공유 기능 완성)

### 지피터스
- [ ] 포스팅 초안 작성 (→ `geekist-post.md`)
- [ ] 스크린샷 4~6장 준비 (지도뷰, 필터, 바텀시트, 공유)
- [ ] 기술 스택 다이어그램
- [ ] 게시 후 댓글 모니터링 (24시간)
- **타이밍**: 화~목 오전 게시 (개발자 활동 시간)
- **목표**: 조회수 500+, 댓글 20+

### 디스콰이엇
- [ ] 프로젝트 등록 (사이드 프로젝트 카테고리)
- [ ] 문제 정의 → 해결 → 지표 중심 서술
- [ ] 지피터스와 동시 게시
- **목표**: 좋아요 50+

### 링크드인
- [ ] 포스팅 초안 작성 (→ `linkedin-post.md`)
- [ ] 스크린샷 2~3장 (지도뷰):  캐러셀 형태로 첨부
- [ ] "문제 → 빌드 → 결과" 스토리 구조
- [ ] 관련 해시태그: #사이드프로젝트 #개발자 #서울카페 #아침루틴 #프로덕트
- [ ] 1촌 네트워크 리포스트 요청 (5~10명)
- **타이밍**: 화~목 오전 8~9시 (출근길 피드 노출)
- **앵글**: "직장인이 출근 전 카페를 찾는 고통 → 주말에 직접 만든 해결책"
- **목표**: 노출 5,000+, 반응(좋아요+리포스트) 100+

### 블라인드
- [ ] "출근 전 카페 가는 직장인" 앵글
- [ ] 강남/여의도/판교 구별 데이터 강조
- [ ] 부담 없는 톤 ("아침 카페 찾다가 직접 만듦")
- **타이밍**: 월요일 아침 또는 금요일 점심
- **목표**: 조회수 1,000+
