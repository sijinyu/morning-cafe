# 채널별 마케팅 체크리스트

## 핵심 메시지
"네이버 지도에서 시간 필터 걸면 프랜차이즈만 나온다 → 프랜차이즈 빼고 개인 카페만 보는 지도를 만들었다"

## 현재 상태
- 웹앱 배포 완료 (morning-cafe-phi.vercel.app)
- iOS 앱 출시 완료 (App Store v1.0.1)
- 지피터스 웹 버전 글 게시 완료
- 링크드인 웹 버전 글 게시 완료

---

## 우선순위 1: 즉시 실행 (이번 주)

### 블라인드 ★최우선 — `blind-post.md`
- [ ] 포스팅 최종본 확인 및 게시
- [ ] 첫 댓글: "강남/여의도 출근하시는 분들 체인 숨기기 한번 써보세요"
- [ ] 댓글 대응 (24시간)
- **타이밍**: 화요일 아침 8시 or 금요일 점심 12시
- **목표**: 조회수 1,000+, 댓글 30+

### 인스타그램 프로필 설정 — `instagram/brand-voice.md`
- [ ] @morningcafeapp 계정 생성/프로필 완성
- [ ] 바이오 링크 설정 (UTM 포함)
- [ ] 앱 스크린샷 5장 캡처 (지도 뷰, 필터, 바텀시트, 사진마커, 리스트)
- **소요**: 30분

### 인스타 강남구 카드뉴스 — `instagram/gangnam-top5-cards.md`
- [ ] Canva 템플릿 제작 (6슬라이드)
- [ ] 캡션 + 해시태그 복붙
- [ ] 게시
- **타이밍**: 블라인드 다음날
- **목표**: 저장 50+

## 우선순위 2: 이번 주 후반

### 링크드인 iOS 팔로업 — `linkedin-ios-followup.md`
