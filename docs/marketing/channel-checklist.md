# 채널별 마케팅 체크리스트

## 핵심 메시지
"네이버 지도에서 시간 필터 걸면 프랜차이즈만 나온다 → 프랜차이즈 빼고 개인 카페만 보는 지도를 만들었다"

---

## 우선순위 1: 즉시 실행

### 지피터스 (사례게시판 > 바이브코딩)
- [ ] 포스팅 최종본 (→ `geekist-vibecoding-final.txt`)
- [ ] 스크린샷 5~6장 삽입
- [ ] 게시 후 댓글 모니터링 (24시간)
- **타이밍**: 화~목 오전 10~11시
- **목표**: 조회수 500+, 댓글 20+

### 디스콰이엇
- [ ] 프로젝트 등록 (사이드 프로젝트 카테고리)
- [ ] 지피터스와 같은 톤, 문제→해결→지표 구조
- [ ] 지피터스와 동시 게시
- **목표**: 좋아요 50+

### 링크드인
- [ ] 포스팅 최종본 (→ `linkedin-post-final.md`)
- [ ] 스크린샷 2~3장 캐러셀
- [ ] 1촌 네트워크 리포스트 요청 (5~10명)
- **타이밍**: 화~목 오전 8~9시
- **목표**: 노출 5,000+, 반응 100+

### 블라인드
- [ ] "출근 전 카페 가는 직장인" 앵글
- [ ] 강남/여의도/판교 구별 데이터 강조
- [ ] "네이버 지도에서 시간 필터 걸면 프랜차이즈만 나와서 만듦" 톤
- **타이밍**: 월요일 아침 또는 금요일 점심
- **목표**: 조회수 1,000+

## 우선순위 2: 1~2주 후

### 인스타그램
- [ ] 계정 생성 (@morningcoffee.seoul 등)
- [ ] 프로필 설정 (링크, 소개)
- [ ] 카드뉴스 1화 게시 (→ `card-news-plan.md`)
- [ ] 해시태그: #아침카페 #새벽카페 #서울카페 #모닝커피 #카페추천 #얼리버드
- [ ] 주 2~3회 게시
- **목표**: 4주 내 팔로워 500

### 네이버 블로그
- [ ] "서울 아침 카페 총정리" 롱폼 포스트
- [ ] 구별 서브 포스트 (강남, 마포, 성수 등)
- [ ] 카페 개별 페이지 URL 링크
- **SEO 키워드**: "서울 아침 카페", "강남 새벽 카페", "6시 오픈 카페"
- **목표**: 네이버 검색 "아침 카페" 1페이지
