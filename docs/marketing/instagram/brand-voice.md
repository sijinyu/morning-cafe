## 브랜드 페르소나

**"아침을 좋아하는 친한 선배"**

- 아침 일찍 카페 가는 습관이 있는 30대 직장인
- 개인 카페 덕후, 프랜차이즈보다 동네 카페
- 정보를 친근하게 공유하는 톤
- 전문가 느낌 없이 "나도 매일 이러는데" 느낌

### 말투 규칙

- **~요 체** 사용 (존댓말이지만 딱딱하지 않게)
- 이모지는 **문장 끝에 1개** 또는 항목 앞에 1개 (남발 금지)
- "ㅋㅋ", "ㅎㅎ" 사용 가능하나 1개 이하
- 느낌표는 게시물당 2개 이하
- 물음표로 공감 유도 ("이런 적 있지 않아요?")

### 톤 예시

```
좋은 예:
"출근 전 30분, 동네 카페에서 조용히 커피 한 잔 하는 시간이 하루 중 제일 좋아요."

나쁜 예:
"☀️✨ 모닝카페와 함께하는 아름다운 아침!! 지금 바로 다운로드하세요!! 🔥💯"
```

## 콘텐츠 믹스 (주 3회 게시 기준)

| 요일 | 콘텐츠 유형 | 예시 |

- 릴스 재공유
- 아침 시간대 실시간 감성 사진 (찍은 게 있다면)
