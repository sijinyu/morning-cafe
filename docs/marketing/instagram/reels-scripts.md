## 릴스 1: "아침 카페 30초 만에 찾는 법"

**길이**: 28초
**형태**: 앱 화면 녹화 + 텍스트 오버레이
**BGM 제안**: 잔잔한 로파이 (CapCut 무료 — "Morning Coffee" 또는 "Sunny Day" 계열)

| 초 | 화면 | 텍스트 오버레이 | 액션 |
|----|------|----------------|------|
| 0-3 | 네이버 지도 "아침 카페" 검색 결과 | "아침 카페 검색하면" | 네이버 지도에서 검색 |
| 3-6 | 프랜차이즈만 나오는 결과 | "프랜차이즈만 나옴" | 스타벅스, 메가커피 강조 |
| 6-8 | 모닝카페 앱 열기 | "그래서 이거 씀" | 앱 아이콘 탭 → 앱 열기 |
| 8-12 | 지도 전체 뷰 | "서울 아침 카페 9,500곳" | 지도 줌아웃 → 마커 전체 |
| 12-16 | 시간 필터 탭 | "6시? 7시? 원하는 시간대" | 6시 → 7시 필터 전환 |
| 16-19 | 체인 숨기기 토글 | "체인 빼기 한 탭" | 토글 ON |
| 19-22 | 구 필터 선택 | "강남구만 볼 수도 있음" | 강남구 선택 |
| 22-25 | 카페 마커 탭 → 바텀시트 | "사진, 메뉴, 별점까지" | 바텀시트 올라오는 장면 |
| 25-28 | CTA 화면 | "모닝카페 — 프로필 링크" | 앱 로고 + @morningcafeapp |

### 녹화 팁
- iPhone 화면 녹화 (설정 > 제어 센터 > 화면 기록)
- 앱 동작은 천천히, 각 단계에서 1초 정도 멈추기
- 세로 모드로 녹화 (릴스 9:16)
- 알림, 상태바 가리기 (CapCut에서 크롭)

### CapCut 편집
- 텍스트: 중앙 상단, 굵은 폰트, 흰색 + 그림자
- 전환: 컷 전환 (fancy 효과 없이)
- 속도: 1x (빠르게 하지 않기)
- 엔딩: 0.5초 페이드 아웃

## 릴스 2: "서울에서 새벽 5시에 여는 카페들"

**길이**: 25초
**형태**: 앱 화면 녹화 + 데이터 텍스트 오버레이
**BGM 제안**: 약간 놀라운 느낌 (CapCut 무료 — "Wait What" 또는 "Did You Know" 계열)

| 초 | 화면 | 텍스트 오버레이 | 액션 |
|----|------|----------------|------|
| 0-3 | 검은 화면 or 새벽 도시 사진 | "서울에서 새벽 5시에 여는 카페가 있다고?" | 텍스트만 |
| 3-7 | 모닝카페 앱 — ~6시 필터 선택 | "ㅇㅇ 있음" | ~6시 필터 탭 |
| 7-10 | 지도에 마커들 표시 | "서울 전역에 이렇게 많음" | 줌아웃하며 마커 전체 |
| 10-13 | 강남구 필터 → 카페 리스트 | "강남구만 1,146곳" | 리스트 뷰 스크롤 |
| 13-16 | 달리는커피 강남논현점 상세 | "이 카페는 새벽 5시 오픈" | 바텀시트 상세 |
| 16-19 | 사진 마커 줌인 | "확대하면 사진도 보임" | 줌인 → 사진 마커 |
| 19-22 | 찜 + 알림 설정 | "찜하면 오픈 전에 알림" | 하트 탭 → 알림 뱃지 |
| 22-25 | CTA 화면 | "모닝카페\n프로필 링크에서 다운" | 앱 로고 + @morningcafeapp |

### 녹화 팁
- 첫 3초가 가장 중요: "새벽 5시에 여는 카페?" — 호기심 유발
- 달리는커피 실제 오픈 시간 05:00으로 확인 후 녹화
- 사진 마커는 줌 레벨 3 이하에서 보이므로 충분히 확대

### CapCut 편집
- 텍스트: 상단 1/3 영역, 굵은 폰트
- "ㅇㅇ 있음" 부분에 약간의 확대 효과 (1.1x, 0.3초)
- 엔딩: 앱 로고 정지 1초

## 공통 릴스 제작 체크리스트

- [ ] iPhone 화면 녹화 ON (제어 센터)

- [ ] 세로 모드 (9:16)
- [ ] CapCut에서 상태바 영역 크롭

- [ ] 릴스 설명에 캡션 + 해시태그 추가
